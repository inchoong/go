夜间模式 By JavaScript

[ 2013-12-27 ] [ 回到首页 ]

夜间模式说简单些就是在页面上增加一个半透明的遮罩层，但是普通遮罩层会挡住页面元素，研究了一下，解决方案：添加 div，给 div 的 outline 属性一个很大的 outline-width 值，用 outline 的边框作为遮罩，这样既能正常与页面元素交互，又能达到夜间模式的效果~

夜间模式

<style>
div {
    position:fixed;
    top: 0;
    left: 0;
    outline:5000px solid rgba(0, 0, 0, 0.3);
    z-index: 99999;
}
</style>
<div></div>

夜间模式 PLUS

<script>
    var brightness;
    // 显示遮罩
    function cover(brightness) {
        if (typeof(div) == 'undefined') {
            div = document.createElement('div');
            div.setAttribute('style', 'position:fixed;top:0;left:0;outline:5000px solid;z-index:99999;');
            document.body.appendChild(div);
        }
        div.style.outlineColor = 'rgba(0,0,0,' + brightness + ')';
    }
    // 事件监听
    window.addEventListener('keydown', function (e) {
        if (e.altKey && e.keyCode == 90) { // Alt+Z：打开夜间模式
            cover(brightness = 0.3);
        }
        if (e.altKey && e.keyCode == 88) { // Alt+X：关闭
            cover(brightness = 0);
        }
        if (e.altKey && e.keyCode == 38) { // Alt+↑：增加亮度
            if (brightness - 0.05 > 0.05) cover(brightness -= 0.05);
        }
        if (e.altKey && e.keyCode == 40) { // Alt+↓：降低亮度
            if (brightness + 0.05 < 0.95) cover(brightness += 0.05);
        }
    }, false);
</script>

还可以写成 GreaseMonkey 脚本，作为浏览器扩展给任意页面增加夜间模式~
