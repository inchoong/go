GNU Wget 1.21.4 for Windows | GNU Wget 的 Windows 二进制文件

命令行实用工具 使用 HTTP、HTTPS 和 FTP 协议检索文件。

警告：某些防病毒工具将 wget-1.21.4-win32.zip 识别为潜在 危险。触发警告的文件是 wget.exe.debug，其中包含 wget.exe 的调试符号，而不是 甚至是可执行文件。如果您的AV给您带来了麻烦，并且您不需要文档或调试符号，则可以下载 WGET.exe直接，或切换到不太损坏的安全产品。

所有二进制文件都是静态编译的，这意味着 wget.exe 不需要任何其他文件即可工作。
