问题：visual studio code怎么改成中文？ 

方法：

首先打开visual studio code，使用快捷键“Ctrl+Shift+X”； 

然后在左侧“扩展”视图文本框中输入“Language Packs”， 

选择需要的中文语言插件进行安装， ( 比如：ms-ceintl.vscode-language-pack-zh-hans ) 

安装完成右下角会弹出一个重启按钮，点击重启即可。
