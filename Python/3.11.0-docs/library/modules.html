导入模块¶

本章中介绍的模块提供了导入其他Python模块和挂钩以自定义导入过程的新方法。

本章描述的完整模块列表如下：

  • zipimport --- 从 Zip 存档中导入模块

    • 例子
  • pkgutil --- 包扩展工具
